# Exercicios-Java
Exercícios simples feitos usando a linguagem Java

* Exercício 1 - Idioma
Programa simples que imprime o idioma do sistema operacional no console.
Arquivo: Idioma.java

CONCEITOS UTILIZADOS:
-Importação de Classes;
-Uso da classe `Locale`;
-Método `getDefault()`;
-Método `getDisplayLanguage`;
-Saída de dados com o `System.out.println`

------------------------------------------------

* Exercício 2 - Média
Programa simples que imprime no console a média entre 2 valores.
Arquivo: Media.java

CONCEITOS UTILIZADOS:
-Variáveis `float` e `int`;
-Operações aritméticas (soma e divisão);
-Atribuição de valores;
-Saída de dados com o `System.out.println`

------------------------------------------------

* Exercício 3 - Nome e Nota
Programa simples que solicita ao usuário o nome e a nota de um aluno. Depois do recebimento dos dados, o programa imprime no console o nome do aluno e a nota informada.
Arquivo: NomeNota.java

 CONCEITOS UTILIZADOS:
 -Entrada de dados com a classe `Scanner`;
 -Variáveis `String` e `float`;
 -Formatação de saída usando o `System.out.printf`
 
------------------------------------------------

* Exercício 4 - Relógio
Programa simples que imprime o horário atual no console.
Arquivo: Relogio.java

CONCEITOS UTILIZADOS:
-Importação da classe `Date`;
-Saída de dados com o `System.out.println`

------------------------------------------------

* Exercício 5 - Resolução
Programa simples que obtém a resolução atual da tela do computador do usuário e exibe a largura e a altura no console.
Arquivo: Resolucao.java

CONCEITOS UTILIZADOS:
-Importação e uso das classes `Dimension` e `Toolkit`;
-Acesso a propriedade de objetos;
-Armazenamento de dados em variáveis;
-Saída de dados usando o `System.out.println`

------------------------------------------------

* Exercício 6 - String para float
Programa simples que converte um valor armazenado como texto (String) para um número decimal (float) e exibe o resultado no console.
Arquivo: StringToFloat.java

CONCEITOS UTILIZADOS:
-Declaração de variáveis;
-Tipo `String`;
-Conversão de tipos;
-Método `Float.parsefloat()`;
-Saída de dados usando o `System.out.println`
